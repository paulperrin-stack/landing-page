# Landing Page

Live demo: [Click here](https://paulperrin-stack.github.io/landing-page/)

A static landing page built from a provided design mockup.

## Built with
- HTML
- CSS (Flexbox)

## What I learned
How to build a multi-section page layout using Flexbox, and how to translate a visual design mockup into working HTML and CSS.
